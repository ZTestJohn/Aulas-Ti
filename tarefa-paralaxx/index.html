Minhas extensões favoritas do VS Code

Essa lista contém minhas extões favoritas ou as que eu julgar importante.

HTML to CSS

Essa extensão faz sugestões das classes e id durante a escrita do css, ela já me ajudou várias vezes além de ganhar tempo.

Live Server

Essa extensão chega a ser essencial para o desenvolvimento web, ela é responsável por atualizar a página web com o html automaticamente. Ela também acaba poupando muito tempo já que não é necessário ficar constantemente clicando para recarregar a página.

Midnight City e Lilac

Essas são as duas extensões de tema que eu uso, elas mudam as cores padrões do Vs Code (Feias), para cores mais marcantes. O lilac contém tons de Roxo/Lilás e Midnight City alguns tons de vermelho, roxo e amarelo.

Keypress Counter

Essa extensão serve unicamente para contar os cliques das teclas do teclado. Eu apenas quero ver o número que irá ficar em alguns anos…

Prettier

Essa extensão é extremamente importante para a organização do código. O Prettier é um formatador de código que auxilia na organização durante o desenvolvimento.
